Лабораторная работа №6
======================

Краткое описание
----------------
Добавление системы пользователей в блог: регистрация, авторизация и использование встроенных механизмов Django для аутентификации.

Реализовано
-----------
- подключено приложение `django.contrib.auth` и модель `User`
- реализована регистрация через создание пользователя `User.objects.create_user(...)`
- обработка случая, когда пользователь с таким логином уже существует
- добавлена форма авторизации: проверка логина и пароля через `authenticate`
- вход пользователя в систему с помощью `login(request, user)`
- шаблоны `registration.html` и `login.html` с HTML-формами и `csrf_token`
- проверка авторизованности пользователя в представлениях (например, при создании статьи)

Запуск
------
cd lab6/blog
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
