Лабораторная работа №4
======================

Краткое описание
----------------
Страница отдельного поста и доработка блога: маршрутизация по id статьи, отдельный шаблон и базовая вёрстка обеих страниц.

Реализовано
-----------
- добавлен URL для просмотра статьи по id (например, `/article/5/`)
- создано представление `get_article`, получающее статью по `Article.objects.get(id=article_id)`
- обработка ошибки `Article.DoesNotExist` через возбуждение `Http404`
- добавлен шаблон `article.html` для отображения одной статьи
- доработан шаблон `archive.html` (список статей) и добавлены ссылки на отдельные страницы
- вынесены общие стили в `article.css` и подключены к обоим шаблонам

Запуск
------
cd lab4/blog
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
