Лабораторная работа №8
======================

Краткое описание
----------------
Использование jQuery для работы с DOM: плавные эффекты при наведении на посты и доработка поведения кнопки «Свернуть».

Реализовано
-----------
- подключена библиотека jQuery через статические файлы (`jquery.js`)
- реализован скрипт `highlight-post.js` с обработкой события `hover` на блоках `.one-post`
- анимация элемента `.one-post-shadow` с использованием `animate` (изменение `opacity` и смещения)
- доработка логики сворачивания постов с использованием jQuery-селекторов и методов (`$(...), .find(...)`)
- демонстрация отличий между `event.target` и `event.currentTarget` в обработчиках
- использование цепочек вызовов и анимаций jQuery для улучшения визуального оформления архива

Запуск
------
cd lab8/blog
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
