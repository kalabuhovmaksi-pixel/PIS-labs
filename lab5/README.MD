Лабораторная работа №5
======================

Краткое описание
----------------
Добавление формы создания новой статьи в блог: обработка GET/POST-запросов, валидация данных и редирект на созданный пост.

Реализовано
-----------
- добавлен URL для создания статьи (например, `/article/new/`)
- создано представление `create_post`, обрабатывающее методы GET и POST
- проверка, что пользователь авторизован, иначе генерация `Http404`
- получение данных формы из `request.POST` (поля `title` и `text`) и простая валидация
- создание объекта `Article` с привязкой к текущему пользователю (`author=request.user`)
- редирект на страницу только что созданной статьи после успешного сохранения
- шаблон `createpost.html` с HTML-формой, полем для заголовка и текстом, а также `csrf_token`

Запуск
------
cd lab5/blog
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
